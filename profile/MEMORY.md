## 観測対象の癖（松永悠 #5035-yu-pdca / 2026-06-11〜06-22）
- 報告フォーマットはほぼ規約通り。朝スケジュール「YYYY年MM月DD日（曜）の予定です」＋時間ブロック、開始/終了は3行、終業「本日の業務を終了します」＋工数表リンク、最後に「本日のノートです」＋画像。
- **種別ラベルが揺れる**: 「SNS：ハイジ」と「SNS投稿：ハイジ」が混在＝同一種別。相場が割れるので正規化候補（§3.5 ラベル品質）。
- **朝スケジュールの日付を誤記しがち**: 06-22 の投稿本文が「2026年06月19日…の予定です」（前日コピペ）。日付は投稿日で補正する。
- **種別なしタスクが多い**: 「求人の更新」「OMOTENASHI」「イハシ移行作業」等は種別：案件 形式を取らない。相場では種別=? に入る。
- **1時間ルールの進捗報告が薄い**: 開始後70〜150分無音がほぼ毎日ある。65分無音検知が日に3〜5回鳴り得る＝本物のシグナル（閾値/静音時間帯は戸田さんと答え合わせ）。
- **タスク途中で別タスクへ切替**しがち（開始の終了宣言なしに次を開始）＝§3.7 突合失敗の主因。base_name フォールバックで多くは拾えるが、残りは1回だけ確認。
